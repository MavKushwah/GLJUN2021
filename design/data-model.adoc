=== Api Contract: Register User

'''

[source]
----
Request: POST /register
Body : {
    "type" : "user",
    "name" : "JohnDoe"
}

Response: JSON
{
    "user_id": "string",
    "secret": "for jwt"
}
----

=== Api Contract: Register Taxi

'''

[source]
----
Request: POST /register
Body : {
    "type" : "taxi",
    "name" : "taxi driver name",
    "taxi_type" : "taxi class",
    "license": "unique number"
}

Response: JSON
{
    "taxi_id": "string",
    "secret": "for jwt"
}
----

=== Api Contract: Create Ride

'''

[source]
----
Request: POST /createride
Header:
 X-User-Id: <Id>
 X-Token: <JWT>
Body : {
    "type" : "taxi type",
    "location" : [<float>, <float>]
}

Response: JSON
{
    "ride_id": "string",
    "topic": "topic name to subscribe for ride update",
    "host": "mqtt host name"
}
----

=== Api Contract: Find Taxi

'''

[source]
----
Request: POST /findtaxi
Header:
 X-User-Id: <Id>
 X-Token: <JWT>
Body : {
   "ride_id": "string"
}

Response: JSON
{
    "success": <boolean>
}
----

=== Api Contract: Login | Logoff

'''

[source]
----
Request: POST /login | POST /logoff
Header:
 X-Taxi-Id: <taxi_id>
 X-Token: <JWT>

Response: JSON
{
    "taxi_id": "string",
    "host": "mqtt host to subscribe",
    "topic" : "topic to subscribe until logoff"
}
----
